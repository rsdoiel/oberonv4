
# Why this fork?

This fork was created so I don't loose track of where to find things related to Oberon System and Language. It is not a fork that is maintained.

I recieved an issue #1 in this repo about issues with the bootloader not compiling. If you trying to do that you might check with [btreut](https://github.com/btreut),
he seems have made some progress for 64bit systems with the help of a large language model. See <https://github.com/btreut/Oberon4Linux-Bootstrap>.
